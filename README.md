## Requirements
 - Python 3.6+

## Installation 
 ```sh
pip install lyricli==0.0.9
 ```
## Usage

```sh
lyricli lyrics [-t, --title] {MUSIC-TITLE} [-r, --artist] {ARTIST-NAME}
```
