# Lyricli

Tmux 키고 터미널에서 작업하고 있는데 현재 틀어논 노래의 가사가 자주 궁금해져서 만들어봤습니다.
코로나 때문에 코노도 자주 못가는데 집에서라도 불러야죠. 🔥

터미널에서 원하는 가사를 보여줍니다.

## Requirements
 - Python 3.6+

## Installation 
 ```sh
  pip install lyricli
 ```
## Usage

```sh
 lyricli lyrics [-t, --title] {MUSIC-TITLE} [-r, --artist] {ARTIST-NAME}
```
